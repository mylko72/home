Vim

Vim 에디터는 유닉스 세상에서 오랜 전통을 이어온 Vi 에디터의 진보된 형태이다. Vi와 Vim은 태생이 유닉스 환경이다보니, 텍스트 환경인 터미널에서 사용되도록 디자인되어 있다. 메모장과 유사해 보이는 Vim 에디터의 첫인상은 화려하고 막강한 기능으로 무장된 요즘 윈도우 기반의 에디터를 접해 봤다면 황당하고 당혹감 마저 느낄 것이다. 하지만 그래픽 환경보다 실행 속도가 수십 배는 빠르고 용량도 작으며 성능도 뛰어나다면 뭔가 허전해 보이는 Vim이 가지고 있는 매력이 궁금하지 않는가??

다음은 Vim 에디터이 특징이다.

  • - 대부분의 운영체제를 지원한다.
  • - 매우 작고 빠르다. (낮은 사양에서도 잘 작동한다.)
  • - 텍스트 환경인 터미널이나 콘솔에서도 작동한다.
  • - GUI 환경도 지원한다.
  • - 프로그래밍 언어별 특수 기능을 지원한다.
  • - 플러그인에 따라 다양한 확장 기능을 지원한다.
  • - 내장 스크립트로 원하는 기능을 만들 수 있다.(커스터마이징 등)
  • - 외부 명령어와 연동할 수 있다.
  • - 작업 녹화나 자동화, 배치 기능을 제공한다.

Vim 명령어

Vim은 일반적으로 마우스를 쓰지 않으므로, 키보드로 범위를 지정하거나 삭제, 복사, 붙이기등 모든 명령어를 단축키로 실행한다. Vim을 능숙하게 사용하기 위해서는 아래의 명령어 단축키와 이동 단축키등을 손가락이 기억할 수 있도록 training을 할 필요가 있다.

❚ 입력하기

Vim은 일반모드(normal mode)라는 상태로 시작되는데 입력을 하려면 입력모드(insert mode)로 전환해야 한다. 일반 모드에서 입력 모드로 전환하는 키는 i 이다. 입력 모드로 전환되면 화면 맨 아래에 현재 모드를 알려주는 '-INSERT-'가 표시되며 이후로는 모든 키가 입력된다. 그외 입력 모드로 전환하는 키는 아래와 같다.

입력모드 전환 키
명령어 설명
a, A a(append)는 현재 커서 위치에서 한 칸 뒤로 이동한 후 입력 모드로 전환된다.
A는 현재 행의 끝으로 이동한 후, 입력 모드로 전환된다.
i, I i(insert)는 현재 커서 위치에서 입력 모드로 전환된다.
I는 현재 행의 맨 앞으로 이동 후, 입력 모드로 전환된다.
o, O o(open line)는 현재 행 아래에 새로운 행을 하나 만든 후 입력 모드로 전환된다.
O는 현재 행 위에 새로운 행을 하나 만든 후 입력 모드로 전환된다.
R 수정(replace) 모드로 작동하므로 모든 글자는 덧쓰여진다.

❚ 저장하기

입력한 내용을 저장하려면 명령행 모드(command line mode)에서 :w를 누르고 엔터를 누르면 파일이 저장된다. 명령행 모드는 일반모드에서 콜론(:)을 누르면 명령행 모드로 전환된다. 또한 다른 이름으로 저장하려면 :w 파일이름으로 입력하면 다른 이름으로 저장할 수 있다.

❚ 종료하기

Vim을 종료하는 명령어는 명령행 모드에서 :q로 명령한다. 또한 저장후 종료하려면 :wq라고 입력해도 된다.

❚ 커서 이동하기

＊ 상하좌우로 이동하기

Vim에서 커서 이동 영문키는 j키를 중심으로 h, j, k, l이며, 주의할 점은 꼭 소문자로 써야 한다. 상하좌우로 이동하는 키의 경우에는 앞에 숫자를 입력해서 반복 횟수를 지정할 수 있다. 예를 들어 j는 한칸 아래로 이동하는데, 10j라고 하면 열 칸을 아래로 이동한다. 그리고 ^은 공백을 제외하고 현재 행의 내용이 처음 등장하는 곳으로 이동한다. $는 현재 행의 맨 끝으로 이동한다.

＊ 화면 스크롤하기
이동 단축키 설명
Ctrl-b 위로 한 화면 스크롤
Ctrl-f 아래로 한 화면 스크롤
Ctrl-u 위로 1/2 화면 스크롤
Ctrl-d 아래로 1/2 화면 스크롤
＊ 문서 특정 위치로 이동하기
이동 단축키 설명
[#]gg #행으로 이동한다. #이 생략되면 첫행으로 이동한다.
[#]G #행으로 이동한다. #이 생략되면 마지막 행으로 이동한다.
:# #행으로 이동한다.
＊ 현재 위치 확인하기

현재 행의 위치 정보를 보기 위해서는 Ctrl-g를 누르면 하단 상태 바에 표시한다.

❚ 삭제, 복사, 붙이기, 취소하기

＊ 삭제하기
명령어 설명
x 커서에 위치한 문자 삭제 (Delete키와 같음)
dd, :d 현재 행을 삭제
[#]dd #행 만큼 줄을 삭제한다. 예를 들어 3dd라고 하면 세 줄을 삭제한다.
D 현재 컬럼 위치에서 현재 행의 끝부분까지 삭제 (d$와 동일)
J 아래 행을 현재 행의 끝에 붙임 (아래 행의 앞부분 공백은 공백 하나로 치환)
gJ 아래 행을 현재 행의 끝에 붙임 (아래 행의 앞부분 공백은 그대로 유지)
＊ 붙여넣기

Vim에서는 삭제 기능이 무조건 잘라내기로 작동하며 삭제된 내용은 임시 저장소 역할을 하는 레지스터에 저장된다. p 명령어는 삭제된 내용을 붙여 넣는다.

명령어 설명
p, :p 현재 행에 붙여 넣는다. 개행 문자가 포함된 경우에는 현재 행의 아래에 붙여 넣는다.
[#]p p명령을 #만큼 반복한다. 만약 5p라고 하면 p 명령이 다섯 번 반복된다.
P 현재 행의 위쪽에 붙인다.
＊ 복사하기
명령어 설명
yy, :y 현재 행을 레지스터에 복사한다.
[#]y 3yy라고 하면 현재 행을 기준으로 세 행을 복사한다.
