Popup Guide
  1. 호출 버튼에 data-ui="popup" 속성을 추가한다.
  2. 호출 버튼 href값에 호출할 팝업의 id값을 넣어준다.
  3. 호출될 팝업 영역에 id값과 data-popup="cont" 속성을 추가한다.
  4. 호출될 팝업 의 닫기버튼에 data-popup="close" 속성을 추가한다.
  5. 호출될 팝업내의 dimmed 마크업이 있다면 해당 마크업에 data-popup=dimmed 속성을 추가한다.
  6. 호출될 팝업내의 dimmed 마크업이 없다면 스크립트에서 자동으로 추가된다.
  7. dimmed 자동 추가가 없어야 한다면 호출 버튼에 data-popup-dimmed="false" 속성을 추가한다.
